# Lista 6

[1. Crie um algoritmo que leia 10 números inteiros e os armazene em um vetor. Em seguida, exiba todos os valores na ordem em que foram digitados.](/Lista6/src/ex1.c)

[2. Altere o programa anterior para que, após a exibição dos valores na ordem em que foram digitados, apresente a média aritmética dentre todos os valores lidos, o maior e o menor dos valores lidos. Utilize funções para isso.](/Lista6/src/ex2.c)

[3. Crie um algoritmo que leia um valor inteiro n, qualquer. Após isso, leia n valores inteiros e apresente todos os valores pares lidos.](/Lista6/src/ex3.c)

[4. Altere o programa anterior para separar os valores entre dois vetores, sendo um vetor para os números pares e outro para os ímpares. Não devem haver posições vazias nos vetores criados. Ao fim, apresente os dois vetores criados. Use funções!!!](/Lista6/src/ex4.c)

[5. Crie um programa que leia um número inteiro qualquer, conta e apresenta quantos dígitos de cada número foram utilizados para formar este número. Por exemplo, 13982 possui a contagem de dígitos 0=0, 1=1, 2=0, 3=1, 4=0, 5=0, 6=0, 7=0, 8=1, 9=1. Seja elegante!!!](/Lista6/src/ex5.c)

[6. Crie um programa que preencha um vetor de 10 posições com valores inteiros diferentes de zero e um valor inteiro X qualquer. Após isso, procure e troque todas as ocorrências de X no vetor por zero. Ao fim, apresente o vetor.](/Lista6/src/ex6.c)

[7. Altere o programa anterior para que X seja também um vetor, mantendo a mesma funcionalidade do enunciado anterior.](/Lista6/src/ex7.c)

[8. Crie um programa que preencha um vetor de inteiros com 15 posições e que leia um valor X inteiro representado uma das posições do vetor. Ordene o vetor em ordem crescente à esquerda de X e decrescente à sua direita.](/Lista6/src/ex8.c)

[9. Crie um programa que leia uma cadeia de caracteres (string) e descubra se ele forma um palíndromo. Exemplo: arara, osso. Desconsidere caracteres especiais, como acentuação gráfica.](/Lista6/src/ex9.c)

[10. Crie um programa que leia uma string qualquer, calcule e imprima a contagem de vogais utilizadas na string.](/Lista6/src/ex10.c)

[11. Crie um programa que leia uma palavra qualquer e criptografe a mensagem antes de apresentá-la, segundo as regras: as vogais são trocadas por (aA = 4, eE=3, iI=1, oO=0, uU= u). Ao fim, apresente a mensagem criptografada.](/Lista6/src/ex11.c)

[12. Crie um programa que leia uma string qualquer e apresente o seu tamanho (quantidade de caracteres, incluindo espaço em branco, desconsiderando ‘\0’). Não utilize biblioteca string.](/Lista6/src/ex12.c)

[13. Crie um programa que leia uma frase e uma palavra a ser buscada nela. Procure e informe se a palavra foi encontrada na frase ou não. Não utilize a biblioteca string.](/Lista6/src/ex13.c)
